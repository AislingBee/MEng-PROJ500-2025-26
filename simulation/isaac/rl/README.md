# Reinforcement Learning

This directory contains the reinforcement learning framework used to train
bipedal locomotion policies for the humanoid robot.

The RL system interfaces with the simulation environment and is designed to
support iterative reward shaping, curriculum learning, and sim-to-real
transfer.

## Scope
- Environment definition
- Observation and action spaces
- Reward functions
- Training and evaluation pipelines
- Policy export for deployment
