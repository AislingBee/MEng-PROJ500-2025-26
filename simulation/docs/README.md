# Simulation & Reinforcement Learning

This directory contains all simulation and reinforcement learning work for the humanoid robot project.

The simulation environment is built using NVIDIA Isaac Sim and is used to:
- Model the humanoid robot and its sensors
- Simulate walking on flat and difficult terrain
- Train reinforcement learning policies for bipedal locomotion
- Support sim-to-real transfer onto the physical robot

All detailed documentation is maintained in Notion to support rapid iteration and collaborative development.  
This repository provides a stable and version-controlled entry point.
